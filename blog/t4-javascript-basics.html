Objects vs Arrays in Accessing Data

Object var pet={ name: ‘Ginger’, breed: ‘toy poodle’, treat: ‘blueberry’ };

To access the values on this object, you can use “dot notation”, that is, the name of the object, a dot, and the name of the property.

console.log(pet.name); //Ginger
console.log(pet.treat); //blueberry

Array var fruitList=[ ‘apple’, ‘orange’, ‘banana’ ];

To access the values, you use a numerical index:

console.log(fruitList[1]); //orange

What is function and why there are useful?

function is a set of codes that perform a specific task in javascript. Once a function is written, it can be called multiple times from within the script. This avoids duplicated codes and also reduce the amount of code that has to be written from scratch each time.

Because function reduce code redundancy, there's less code to be tested. Also because functions are self-contained, once we’ve tested a function to ensure it works, we don’t need to test it again unless we change it. This reduces the amount of code we have to test at one time, making it much easier to find bugs.
