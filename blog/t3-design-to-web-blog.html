Design to Web

Responsive Website

Responsive Website is a website that will scale itself to display properly across different devices. It's important because there is more and more people viewing the website on a mobile or tablet, having a good responsive design with fluid display will save the trouble of having to redesign the website to support newer devices and screens in the future.

Mobile First Design

Start your design process with mobile considerations and constraints and then work your way up from there. A website optimized for mobile will load faster, get better ranking at Google and save bandwith cost.

Frameworks

Framework is a collection of codes (html, css, JS) that coders can used in their projects that will speed up their development as they don't have to write everything from scratch.

Advantages:

  • speeds up the development process.
  • solution to common css problems.
  • browser compatibility.

Disavantages:

  • unused code leftover.
  • slower learning curve.
  • you don't learn to do it yourself

Wireframes

Wireframe is about the functionality of a website.It can be a really simple sketch that demonstrates what sort of things you can do in your design. For example, a wireframe of a website will show the navigation, the main buttons, the columns, the placing of different elements. You can think of it as a blueprint for a website.
