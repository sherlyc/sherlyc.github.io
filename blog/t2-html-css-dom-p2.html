HTML, CSS, DOM #2

What is Grid Based Design?

A grid-based design is one that works in and around the constraints of a row-and-column-based grid system to help keep web content nicely aligned.

The human eye 'sees' a web page a certain way, roughly from the top left to the bottom right, the grid will guide the user eyes to see elements in a pleasing and distinctive way.

What's all the hype about responsive webpages?

With the increased users on smart phones and tablets, this is a logical solution for those who do not want to invest in various versions of a website for all those different devices.

A responsive website is designed based on a flexible grid and it would scale automatically to fit accordingly to any screen resolution.

Depending on the screen resolution, some contents can be positioned accordingly for easier navigation

What is semantic structure?

Semantic structure means writing your website in a clean and descriptive html markup tags.

Many visual impaired people rely on text to speech program to read webpages back to them, semantic code aids accessibility.

It also helps to improves the ranking of a website in a search engine as it is easier for the "search engine spiders" to understand your contents.
