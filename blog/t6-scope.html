Javascript Scope

What is Scope?

In Javascript, scope refers to the current context of code. It's where your functions or variables are assessible, be it Global or local scopes.

Global Scope

Variables when defined globally can be accessed across the scopes. For example, this name variable which is declared global, can be accessed from any part of the script.

//global variable var name="James";

Local Scope

Local scope is any scope defined past the global scope. Usually, there is a global scope and each function defined has its own local scopes. A variable defined in a function is only available to the local scope and unavailable to outside of scope.

//global variable var name="John"; function displayName(){ var name="James"; console.log("name"); //James } console.log("name"); //John
